# 42 Ecole Exams Repository

This repository has been created to organize and store my 42 Ecole exams. Each exam is located in its respective folder and contains the source code and explanations.

## Folder Structure

- **Rank_2/**
  - **first_word/**: Contains the `first_word.c` file and a `README.md` for explanations.
  - **ft_list_remove_if/**: Contains the `ft_list_remove_if.c` file and a `README.md` for explanations.
  - **print_bits/**: Contains the `print_bits.c` file and a `README.md` for explanations.
  - **str_capitalizer/**: Contains the `str_capitalizer.c` file and a `README.md` for explanations.

## Purpose

The purpose of this repository is to keep the code and solutions I wrote during exams in one place, allowing me to reference them in the future and share them with others.

## Usage

Each folder contains the solution to the respective exam question. For more information, you can check the `README.md` files in the respective folders.
